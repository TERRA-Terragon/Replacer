==========Содержание==========

1) Запуск готового исполняемого файла
2) Структура проекта

==========1 Запуск готового Исполняемого файла==========
Для запуска готового исполняемого файла вам достаточно всего лишь
запустить replacer.exe удобным для вас способом

ВНИМАНИЕ:
Данный файл был создан на устройстве с архитектурой "x86_64" и ОС "windows"
при несовпадении с вашими, готовый исполняемый файл запуститься не сможет из-за особенностей устройства. В случае неполадок следует обратиться к разработчику или прибегнуть к пункту 3 "Сборка исполняемого файла" так как код абсолютно рабочий для каждого
устройства, однако исполняемые файлы варьируются в зависимости от устройства, ручная сборка
поможет создать файл конкретно под ваше на любой операционной системе и на любой архитектуре

==========2 Структура проекта==========
Проект написан с использованием языка программиравания "Python"
из-за наличия большого количества инструментов у данного языка для подобных программ

в данном проекте текст написаный после символа '#' является комментарием
и не влияет на работу программы

Проект состоит из основного файла "main.py", дополнительного с описанием класса
"Replacer" файла "replacer.py" и готового исполняемого файла "replacer.exe"

main.py:

1) создание и управление графическим окном
2) чтение текста из файлов

replacer.py:

1) замена персональных данных в тексте
