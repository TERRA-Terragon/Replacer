2) Структура проекта
3) Ручная сборка проекта

1 Запуск готового Исполняемого файла
Для запуска готового исполняемого файла вам достаточно всего лишь
запустить replacer.exe удобным для вас способом

ВНИМАНИЕ:
Данный файл был создан на устройстве с архитектурой "x86_64" и ОС "windows"
при несовпадении с вашими, готовый исполняемый файл запуститься не сможет из-за особенностей устройства. В случае неполадок следует обратиться к разработчику или прибегнуть к пункту 3 "Сборка исполняемого файла" так как код абсолютно рабочий для каждого
устройства, однако исполняемые файлы варьируются в зависимости от устройства, ручная сборка
поможет создать файл конкретно под ваше на любой операционной системе и на любой архитектуре

2 Структура проекта
Проект написан с использованием языка программиравания "Python"
из-за наличия большого количества инструментов у данного языка для подобных программ

в данном проекте текст написаный после символа '#' является комментарием
и не влияет на работу программы

Проект состоит из основного файла "main.py", дополнительного с описанием класса
"Replacer" файла "replacer.py" и готового исполняемого файла "replacer.exe"

main.py:

1) создание и управление графическим окном
2) чтение текста из файлов

replacer.py:

1) замена персональных данных в тексте

3 Ручная сборка проекта
Данная инструкция предоставления для ОС "Windows"

Для сборки проекта вам необходим интерпритатор языка программирования "Python".
вы можете проверить его наличие введя в консоль следующую команду
python --version
если вывод подобен этому: Python 3.14.0 интерпритатор уже установлен
если вы получили ошибку, вам необходимо его скачать. Делайте это только с официальных источников (просто загуглите "download Python")
или, если вы доверяете ссылкам, можете воспользоваться этой https://www.python.org/downloads/ (официальный сайт)

Сборка:

1) Зайдите в папку с кодом в консоли
    cd "путь к папке с кодом"

2) Создание виртуального окружения на которое будут установлены зависимости:
    python -m venv .venv

3) Активация виртуального окружения:
    2.1) Вы используете CMD:
        .\.venv\Scripts\activate.bat
    2.2) Вы используете PowerShell:
        .\.venv\Scripts\Activate.ps1

4) Установка зависимостей:
    pip install python-docx natasha pyinstaller

5) Сборка в exe файл:
    pyinstaller --onefile --name replacer.exe --noconsole --add-data=".venv\Lib\site-packages\natasha\data\;natasha\data\" --distpath . main.py
    Разберем данную команду:
        5.1) pyinstaller утилита для сборки файлов на языке программирования "Python" в исполняемый .exe
        5.2) --onefile сокращение выходных файлов
        5.3) --name имя выходного файла
        5.4) --add-data="..." добавление данных используемых библиотекой "natasha" в исполняемый файл
        5.5) --dispath . куда сохранить исполняемый файл ('.' - текущая папка)
        5.6) main.py исходный файл
После этого вы получили готовый replacer.exe готовый к исполнению
